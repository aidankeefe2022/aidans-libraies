# AidanDynaArray Interface

## Options

```c++
#define AIDAN_SHORT_NAMES 
#include <LibAidan/AidanAidAllocator.h>
```

this provides Api macros with our aid_ prefix.

## Types
Typed dynamic-array structs generated by macros:
- `aid_DArray_u64`
- `aid_DArray_i64`
- `aid_DArray_f32`
- `aid_DArray_f64`
- `aid_DArray_string`
- `aid_DArray_void`

Each stores `cap`, `len`, and `data`.

## API (Macros)
### Capacity growth: `arr_incr_cap`, `arr_incr_cap_cust`

arr_incr_cap will use stdlib realloc.

arr_incr_cap_cust takes an extra parameter that is a function pointer to a custom realloc implementation that adheres to the stdlib realloc interface.
 
### Push: `arr_push`, `arr_push_cust`

arr_push will call arr_incr_cap to increase array size by 10 slots if the array is too small to hold the new data

arr_push_cust will call arr_incr_cap_cust and takes an extra param that is a function pointer to a custom realloc function that adheres to the stdlib realloc interface.

### Pop: `arr_pop`

arr_pop decrements the array size and returns the value in the last slot.
