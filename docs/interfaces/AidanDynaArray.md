# AidanDynaArray Interface

## Types
Typed dynamic-array structs generated by macros:
- `aid_DArray_u64`
- `aid_DArray_i64`
- `aid_DArray_f32`
- `aid_DArray_f64`
- `aid_DArray_string`
- `aid_DArray_void`

Each stores `cap`, `len`, and `data`.

## API (Macros)
- Capacity growth: `arr_incr_cap`, `arr_incr_cap_cust`
- Push: `arr_push`, `arr_push_cust`
- Pop: `arr_pop`

`aid_` prefixed macro variants are also provided.
